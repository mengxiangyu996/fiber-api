# breeze-api

### 实现 web 快速开发的脚手架

### 使用说明
###### 克隆项目
```
git clone https://github.com/mengxiangyu996/breeze-api.git
```
###### 进入目录
```
cd breeze-api
```
###### 修改配置文件
```
mv env.json.example env.json
```
###### 安装依赖
```
go mod tidy
```
###### 复制依赖到本项目
```
go mod vendor
```
###### 启动服务
```
go run main.go
```
